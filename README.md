# 📌 Overview

Interview Prep Buddy is a full-featured Flutter-based mobile application developed to simplify and modernize interview preparation. The app provides users with curated interview questions, AI-powered answer evaluation, mock interview sessions, performance analytics, real-time notifications, and secure authentication.

The project was developed as a major academic and practical learning project focusing on:

- Flutter App Development
- Firebase Integration
- AI-Based Features
- Real-Time Communication
- Clean UI/UX
- Mobile Security
- Backend Services
- State Management
- Deployment & Optimization

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/interview-prep-buddy.git
```

## 2️⃣ Navigate to Project

```bash
cd interview-prep-buddy
```

## 3️⃣ Install Dependencies

```bash
flutter pub get
```

## 4️⃣ Configure Firebase

- Create Firebase Project
- Add Android App
- Download `google-services.json`
- Place file inside:

```bash
android/app/
```

## 5️⃣ Run Application

```bash
flutter run
```
